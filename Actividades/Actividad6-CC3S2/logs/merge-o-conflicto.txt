# git-merge.txt

## Merge de feature/advanced-feature en master

Se creó la rama `feature/advanced-feature` desde `master` y se añadió la función greet():

def greet():
    print('Hello como una función avanzada')

greet()

Commit en la rama feature/advanced-feature:

En la rama `master`, se realizaron cambios en el mismo archivo `main.py`, modificando el print original a:

print('Hello World-actualiado en main')


### Conflicto generado
Al ejecutar:

$ git merge feature/advanced-feature

Git mostró un conflicto en `main.py`:

<<<<<<< HEAD
print('Hello World-actualiado en main')
=======
def greet():
    print('Hello como una función avanzada')

greet()
>>>>>>> feature/advanced-feature

---

### Resolución
Se decidió **combinar ambos cambios** para mantener la funcionalidad de `greet()` y el mensaje actualizado:

print('Hello World-actualiado en main')

def greet():
    print('Hello como una función avanzada')

greet()

Archivo corregido → guardado y añadido:

$ git add main.py
$ git commit -m "Resuelve el conflicto de fusión entre la versión main y feature/advanced-feature"
